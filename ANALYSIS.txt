Q2: 它的核心功能是什么——如何实现"图文格式带动画效果"
─────────────────────────────────────────────────────

【核心功能流程】:
  1. 用户在左侧输入 Markdown
  2. 右侧实时预览（所见即所得）
  3. 支持数学公式、图表、代码高亮等
  4. 导出为 PNG/PDF/HTML/ZIP

【动画实现】:
  ✅ CSS @keyframes 定义（无需外部库）
  
  • fadeIn (淡入) 
    - 位置: style.css 第 1570-1577 行
    - 触发: posterContent 首次渲染时 (0.5s)
    
  • renderPulse (脉冲)
    - 位置: style.css 第 776-786 行
    - 触发: 配置变更时 (1.5s 循环)

【图文渲染】:
  • HTML 结构: <div class="markdown-poster">
    ├─ <div class="poster-header"> (z-index: 1)
    ├─ <div class="poster-bg-image"> (z-index: 0 - 背景层)
    ├─ <div class="poster-content"> (z-index: 1 - 内容)
    └─ <div class="poster-footer"> (z-index: 1)

Q5: 输出格式是什么
───────────────────

【导出格式】

  1. PNG (单图导出)
     函数: exportToPNG() (script.js 第 2510-2599 行)
     方式: html2canvas → Canvas → toDataURL()
     特性: 支持透明背景, 自动裁剪边缘, 高清输出

  2. PDF (矢量文档)
     函数: exportToPDF() (script.js 第 2601-2717 行)
     方式: html2canvas → jsPDF
     特性: A4 或自适应纸张, 适合打印

  3. HTML (完整网页)
     函数: exportToHTML() (script.js 第 2719-2850 行)
     方式: DOM 克隆 + 样式内联 + 图片 Base64
     特性: 保留所有样式和交互

  4. 多图 ZIP (分页压缩包)
     方式: splitMarkdownIntoPages() + JSZip
     特性: 3:4 比例分页, 自动页码标注

【导出模式】
  • 自由模式  → 480~800px 可调
  • 小红书    → 1080x1440px (3:4)
  • 朋友圈    → 1080x1920px (长图)

【背景系统】
  3 种背景类型:
  
  1️⃣ 渐变背景 (Gradient)
     • 8 款预设 + 自定义渐变
     • CSS: linear-gradient()
     • 支持 6 种渐变方向
     • 实现: 直接应用到 .markdown-poster
  
  2️⃣ 纯色背景 (Solid)
     • 颜色选择器
     • 实现: background: solidColor
  
  3️⃣ 背景图 (Image)
     • 上传图片
     • 虚化程度: 0~30px
     • 透明度: 0~1
     • 实现: 额外 div 层 (.poster-bg-image)
       └─ filter: blur(12px)
       └─ opacity: 0.3
       └─ z-index: 0

【Canvas 导出】(★ 最复杂部分)

  函数: renderWithFallbackScales() (script.js 第 2908-2992 行)
  
  步骤:
  1. 创建导出节点 (克隆 + 配置)
  2. 预处理图片 (跨域、防盗链)
  3. 等待字体加载
  4. html2canvas 渲染
     - 倍数: [1.5x, 1x, 0.5x] (自动回退)
     - 选项:
       • useCORS: true (启用跨域)
       • allowTaint: false (防污染)
       • backgroundColor: null (保留透明)
       • onclone: 特殊处理 KaTeX/Mermaid/SVG
  5. 裁剪透明边缘
  6. canvas.toDataURL() → PNG 数据
  7. 触发下载

【智能分页】(script.js 第 3861-3991 行)

  流程:
  1. 将 Markdown 拆分为原子块
  2. 逐块离屏测量实际高度
  3. 按 maxPageHeight 分割
  4. 保证块完整性 (代码块、图表不拆)
  5. 生成多页预览 (网格 + 轮播)
  6. 多图导出时逐页生成 PNG

✅ 防抖渲染 (debounce)
   每次输入延迟 300ms，避免频繁 DOM 更新

✅ 懒加载库
   html2canvas 和 jsPDF 仅在导出时加载

✅ 多倍数回退
   显存不足时自动降低渲染分辨率

✅ 图片缓存
   LRU 缓存管理 (最多 50 张)

✅ 事件委托
   多图预览使用事件委托而非逐个监听

✅ 草稿自动保存
   配置和内容实时保存到 localStorage
